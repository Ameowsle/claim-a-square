# Claim a Square

A tiny interactive web app: a 10x10 grid of squares. Click one to claim it and
watch the live counter and progress bar fill up.

Open `index.html` in a browser to try it.

## How this was built

A case study in how a real project grows: one feature at a time, each on its own
branch and merged through a pull request. Never straight onto `main`.

See the Pull requests tab (filter: Closed) and Insights -> Network.
